# =============================================================================
# requirements.txt  —  TRAINING / EVALUATION / MODEL environment
#
# Face EXTRACTION dependencies are deliberately NOT here. They live in
# requirements-preprocess.txt and are meant to be installed in a SEPARATE,
# isolated environment (see README, "Why two requirements files"). Reason:
# facenet-pytorch hard-pins torch<2.3 / numpy<2.0, which would drag this whole
# stack back to early-2024 versions; extraction is a one-time job over raw
# videos and does not need the same torch as training.
#
# Install (fresh remote / local CPU dev):
#     pip install -r requirements.txt
#     pip install -e .                 # editable install of the `sfdet` package
#
# CUDA note: on Linux/Colab the default PyPI `torch` wheel is already a CUDA
# (cu12x) build, so the lines below install GPU torch as-is. If you need a
# specific CUDA tag (driver mismatch) or a CPU/Windows build, install torch +
# torchvision FIRST from the PyTorch index, then run the line above:
#     https://pytorch.org/get-started/locally/
# Versions are pinned for reproducibility; bump torch/torchvision together to
# match your Colab runtime if you ever hit a driver/CUDA mismatch.
# =============================================================================
